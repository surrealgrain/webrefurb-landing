# Pilot Plan

## Batch Size

Start with 10 manually reviewed ramen/izakaya leads.

## Before Sending

- Confirm category is ramen or izakaya.
- Confirm email/contact form is an approved route.
- Confirm no do-not-contact, bounced, or stale send-ready state.
- Confirm first-contact copy is QR-first and reply-only.
- Get explicit manual approval.

## During Pilot

- Track replies.
- Track owner questions.
- Track demo confusion.
- Track trial requests.
- Track time required to build each trial.

## Stop Conditions

- High bounce rate.
- Owners misunderstand the product as ordering/payment software.
- Trial pages take too long to build manually.
- Owner confirmation process is unclear.
- Any wrong recipient or wrong public page incident.
